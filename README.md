# 📈 台灣股市財報自動化抓取與資料清洗流程 (Financial Data Automation Pipeline)

這個專案建構了一套自動化的資料管線 (Data Pipeline)，從公開資訊觀測站等管道自動化下載財務報告書 PDF、解壓縮法人報告、抓取財報數據，並最終清洗轉換為結構化的資產負債表與損益表 Excel 檔案，大幅提升金融資料收集與處理的效率。

## 🛠 核心功能與檔案說明

本專案由四個核心 Jupyter Notebook 組成，各自負責資料管線中的不同階段：

1. **`web_crawler.ipynb` (財務報告書 PDF 自動化下載)**
   - 使用 `Selenium` 模擬瀏覽器行為，自動應對公開資訊觀測站的等待與跳轉機制。
   - 根據設定的年份、季度與資料起訖日，自動化批次下載母子公司合併報表 (A02) 或母公司財報 (A01) PDF 檔。

2. **`unzip_file.ipynb` (每日法人報告自動化解壓縮)**
   - 定期動態監測暫存資料夾，根據時間戳記 (例如：20240607 起) 篩選出特定的 `.zip` 壓縮檔。
   - 自動將檔案解壓縮至團隊共享專區（如 OneDrive 產業專區），並在完成後自動移除壓縮檔，釋放硬碟空間。

3. **`web_scraper.ipynb` (資產負債表數據抓取與漏失檢查)**
   - 抓取並產出各個股票代碼的「資產負債表」Excel 檔案。
   - 內建錯誤檢查機制：自動比對目標股票清單與已成功產出的檔案，計算出未成功抓取的缺失代碼，並自動輸出為 `missing_balance_ids.txt`，確保資料完整性。

4. **`data_cleaning.ipynb` (財報資料清洗與格式轉換)**
   - 使用 `Pandas` 與 `glob` 批次讀取各個上市櫃公司的財報 CSV 資料。
   - 透過正則表達式 (`re`) 與字串切片自動提取股票代碼。
   - 整合歷史多年（104年至112年）的損益表與資產負債表數據，將其標準化並匯出為多個結構化的 Excel 報表，方便後續量化分析。

## 📐 系統架構圖 (Data Pipeline Workflow)

```text
[公開資訊觀測站 / 法人報告]
        │
        ▼ (Selenium 爬蟲下載)
 [PDF 財報 / ZIP 壓縮檔] ───> (自動解壓縮模組) ───> [OneDrive 共享資料夾]
        │
        ▼ (Scraper 數據抓取 & 漏失比對)
 [原始財報 CSV / Excel] ───> (檢查未抓取清單) ───> [missing_balance_ids.txt]
        │
        ▼ (Pandas 資料清洗與標準化)
 [最終標準化財報 Excel (104-112年)] ───> 供量化策略、財務分析使用
